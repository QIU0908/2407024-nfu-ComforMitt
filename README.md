更新順序

1. 先轉到主分支(git checkout main)

2. 將他人更新分支下載(git pull)

3. 建立分支(git branch 學號)

4. 從主分支轉換到分支(git checkout 學號)

5. 將本地資源上傳

6. git add .

7. git commit -m"更新內容敘述"

8. 第一次使用或要更改位推送分支

9. git push -u origin 學號

10. 後續幾次若要更新到同樣分支只需(git push)
